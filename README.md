# EV Website

A modern and responsive Electric Vehicle (EV) website built using React and Vite. The website provides users with information about electric vehicles, their benefits, features, and future impact on sustainable transportation.

## Features

- Modern and responsive user interface
- Fast performance with Vite
- React component-based architecture
- Smooth navigation using React Router
- Mobile-friendly design
- Clean and organized project structure

## Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router DOM

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/ev-website.git
```

### Navigate to the Project Directory

```bash
cd ev-website
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

### Open in Browser

```
http://localhost:5173
```
